#### Install Project
```bash
    git clone https://github.com/HanNewbie/Tubes_DPW.git
```
```bash
    composer install
```
```bash
    npm install
```
-  copy env file lalu sesuaikan dengan database
```bash
    php artisan artisan migrate:fresh --seed
```
```bash
    php artisan db:seed
```
```bash
    php artisan storage:link
```
